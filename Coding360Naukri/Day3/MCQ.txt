Which is the correct CSS syntax written inside the style tag?

body { font-size: 20px; }
(A CSS Syntax rule consists of a selector, property, and its value. The selector points to the HTML element where the CSS style is to be applied. The CSS property is separated by semicolons. It is a combination of the selector name followed by the property: value pair that is defined for the specific selector.)


Which CSS property controls the text size?

=> font-size
The CSS property that controls the text size is font-size. You can use the font-size property to specify the size of text within an element. The font-size property allows you to adjust the size of the text in your HTML elements, making it larger or smaller as needed.
