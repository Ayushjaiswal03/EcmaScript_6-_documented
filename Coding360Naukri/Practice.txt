Q- How do you declare a variable in JavaScript?

=> var

It allows you to store data that can be accessed and manipulated throughout your code.

Q- What should be printed on the console for the following JavaScript statement?

var obj = {};
function A() { return obj; }
function B() { return obj; }
console.log( new A() == new B() );

=> true
The statement new A() will make a new object of ‘A’ and call the constructor of object ‘A’, which will return var obj. 
Similarly the statement new B() will make a new object of ‘B’ and call the constructor of object ‘B’, which will return var obj. 
Hence console.log( new A() == new B() ); will print true in the console.
